一旦设定了密码，那么每次登录桌面总是要输入密码才行的。

当然，密码不能太复杂，否则除了脑瓜疼记不住外，电脑也会面目可憎变得不好用了。

图01



要想让它自动登录，必须要在还能进得去的时候动手。

按下 `徽标键+R` ，弹出运行窗口。

图02



那位说徽标键是啥？

徽标键就是 `Win` 键，就是印有 `Windows` 窗口图标的那个按键，就像这样的。

图03



好，在打开的运行窗口中输入 `regedit` ，回车。

这时你会看到，一个叫作注册表编辑器的窗口被打开了。

图04



注册表又是啥玩意？

你只要记住，它是用来存放系统各种参数的一种数据库就行了。

接下来我们就只需要在这里操作就行了。

好，我们定位到如下注册表路径：

```
计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon
```



什么？你不会定位？

很简单啊，只要把上面那个路径复制下来，然后粘贴到注册表编辑器的地址栏内回车即可。

图05



定位OK后，我们就可以找一找下面这几个注册表项了。

* `DefaultUserName` - 自动登录时的用户名
* `DefaultPassword` - 自动登录时的密码
* `DefaultDomainName` - 自动登录时的域名（非域用户可不写）
* `AutoAdminlogon` - 开启自动登录的开关，`1` 开启，`0` 关闭，默认为 `0`



拢共就四个，意思也给你们解释过了，赶快找一找，看谁找得又快又准哈！

图06



什么？只找到了三个？

那个 `DefaultPassword` 没找到？

呃，其实这个没找到没关系的，你自行创建一个也是可以的。



右键点击空白处，选择 `新建` > `字符串值` 。

这个 `字符串值` 也就是注册表项的类型 `REG_SZ` 。

图07



然后将新建的注册表项名 `新值#1` 修改重命名为 `DefaultPassword` 即可。

图08



到现在为止没问题吧？

接下来我们只要将这四个注册表项设定为我们要自动登录的用户信息就行了。

比如，用户名是 `administrator` ，密码是 `123456` ：

```
DefaultUserName: administrator
DefaultPassword: 123456
DefaultDomainName: 可不填写
AutoAdminlogon: 1
```

OK，大功告成！

如果失败了，那么就要回过头来检查一下，你的用户名和密码有没有写对。

顺便说一句，那个 `DefaultDomainName` 是用来指定域用户的，企业域环境用得比较多，一般家用就不用填写了。

最后再掰扯一下，这个自动登录功能可能会给我们带来困扰的几个地方。

首先是安全风险问题，我们保存在注册表的用户密码是明文形式，因此只要是能物理接触到电脑的人都可以直接看到。

所以说从安全角度讲，开启自动登录的电脑最好是放在别人不易接触到的地方，或者是自己家里比较安全的地方。



其次，如果自动登录已经生效，而我又想在开机的时候切换使用其他用户登录，又该怎么办呢？

很简单，你不用去禁用自动登录功能，只要注销当前用户，或者在开机重启时按住 `Shift` 键即可。

这么操作之后，系统就会出现登录界面，你就可以放心大胆地输入密码了。
