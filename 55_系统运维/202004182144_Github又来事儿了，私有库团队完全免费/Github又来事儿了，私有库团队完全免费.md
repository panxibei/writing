其实吧，去年 GitHub 就宣布私有仓库免费，同时提供 3 个免费协作人数的名额。

但如果需要更多的协作人数，则需要付费。

那从现在开始，任何团队都可以免费使用不限制协作人数的私有仓库。

当然了，如果要使用到一些高级功能，肯定还是要付费的。

此次进一步开放免费，不知道其他一些竞争对手作何感想。
