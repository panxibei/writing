`Telnet` 登录

图a01



使用 `PuTTY` 来登录 `Telnet` 。

连接类型 `Connection type` 一定要选择其他 `Other` ，并且右侧框中定位为 `Telnet` 。

图a02



或者直接使用命令行，加个 `-telnet` 参数即可。

```
putty -telnet 192.168.1.123
```



登录显示和直接使用 `Telnet` 类似，不过当然是 `PuTTY` 的窗口了。
